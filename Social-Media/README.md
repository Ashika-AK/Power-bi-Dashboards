
# Social Media Analytics Dashboard

This project contains an interactive Power BI dashboard that provides insights into social media performance.

##  Overview
The dashboard analyzes social media data to help track and understand:
- Audience engagement
- Reach and impressions
- Top-performing posts
- Trends over time

##  Files in this folder
- `Social media analytics dashboard.pbix` → Power BI file for the dashboard  
- `social_media_analysis.csv` → Dataset used for building the dashboard
- Social_media_analytics.png -> Screenshot of the dashboard  


##  Insights
- Identifies which posts get the most engagement  
- Highlights audience behavior patterns  
- Helps optimize social media strategy with data-driven decisions
